The Design Process of this Website

The design of this website went through several iterations. Feedback from each version accentuated refinement of key considerations during the design process. For brevity, only 2 versions are shown here.

Information Architecture

Object Map

Borrowed from Sophia Voychehovski's object oriented UX (OOUX) approach the object map below aims to provide an overview of the site requirements. The map describes the site as represented by categories defined by the user mental model, great for understanding content specifications.

Site map 1.0
Sketch 1.0

Revised Design

After observing user reactions key elements were rethought with the user journey in mind. The portfolio section was made the homepage, redesigned with a focus on retaining attention with hero images, consistency between page layouts and highlighting key tools from each portfolio piece. CV and story were recognized as semantically similar and grouped together simplifying navigation.
